# Work Hub — Daily Dashboard

A personal work-life dashboard with:

- **Dashboard** — daily overview with date, pending expenses, meetings today, task progress, Gmail quick-access
- **Expense Tracker** — log expenses with receipt name, purpose, amount & date; toggle reimbursement status; AI receipt scanning
- **Calendar** — monthly calendar with meeting dots; add/delete meetings; daily 7 AM SGT browser notifications
- **To-Do List** — tasks with progress bar and completion tracking
- **Google Sheets** — link and search your spreadsheets
- **Gmail** — embedded inbox with Inbox / Starred / Sent / Trash tabs

## Running locally

Open `index.html` in any modern browser. All data is saved in `localStorage`.

## Tech

Vanilla HTML + CSS + JavaScript. No build step required.
